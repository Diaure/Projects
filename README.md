# 🎓 Mes Projets Data

Bienvenue sur mon portfolio de projets Data Analyst & Machine Learning !

J’explore des problématiques variées — marché du vin, mobilité, emploi, scoring, prédiction — en mobilisant Python, SQL, Power BI et des techniques de Machine Learning pour transformer des données brutes en insights exploitables et en solutions concrètes.

**Objectif** : Montrer comment la data peut répondre à des enjeux réels, soutenir la prise de décision et s’intégrer dans des applications opérationnelles.<br>
Chaque projet présente le contexte, les données utilisées, les analyses menées, ainsi que les solutions développées (API, dashboards, modèles ML, automatisations…).
